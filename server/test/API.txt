User REST API
  POST /users
    creates a user
  GET /users/me
    returns a user himself
  PATCH /users/me
    modified a user himself
    returns 409 if email is isOccupied
    returns 400 if passwordOld is wrong
  DELETE /users/me
    delete a user himself
  DB Validation(POST, PATCH)
    returns 400 if email already exists
    returns 400 if email not valid
    returns 400 if password less then 4 char

Auth API
  POST /login
    returns success and 200
    returns error and 400 without email
    returns error and 400 without password
    returns error and 401 if email false
    returns error and 401 if password false
  GET /logout
    logout user

Account REST API
  PARAMS
    returns 404 if account not exist
    returns 403 if user not authorized
  POST /accounts/me
    creates account
  GET /account/me
    returns accounts to user
  GET /account/me/:id
    returns single account to user
  PATCH /account/me/:id
    modified account
    returns 409 if account name is isOccupied
  DELETE /account/me/:id
    delete account
  DB Validation(POST, PATCH)
    returns 409 if account name already exists
    returns 400 if account name is empty
